# 💻  Linux Command
```
$ ls : 폴더 내 파일 조회
$ ls -l/etc/www/a* : 목록 중 앞글자가 a인 목록 조회
$ ls -lahS : 숨김파일 포함하여 파일 크기 순으로 용량이 표시되게 조회
$ ls -ult : 파일의 사용시간
$ ls -clt : 파일의 최근 변경 시간
$ history : 사용한 명령어 목록 조회

$ mkdir 이름 : 디렉토리 생성
$ cd 이름 : '이름'을 가진 파일로 이동
$ cd ~ : 최상위권 파일로 이동
$ cd ../ : 이동한 파일의 바로 앞의 상위권 파일로 이동
$ ls : 파일 리스트 출력
$ cal : 달력 출력
$ cal > 이름 : 달력을 '이름'이라는 파일로 저장
$ who : 누가 접속했는지 조회
$ who > 이름 : 접속 정보를 '이름'이라는 파일로 저장

$ cat 이름 : '이름'의 파일 내용 보기
$ cat 이름1 이름2 : '이름1', '이름2'의 파일 내용 동시에 보기
$ cat 이름1 이름2 > 이름3 : '이름1','이름2'의 내용을 '이름3'파일로 생성
$ cat 이름1 >> 이름2 : '이름1' 내용을 '이름2'의 내용 뒤에 붙이기
$ cat -n 이름 : '이름'파일을 행 번호와 같이 출력
$ vi 이름 : '이름'파일의 내용 보기(i눌러서 수정 가능)

$ tac 이름 : '이름'의 파일 내용을 역으로 보기
$ tee : 명령어의 출력결과를 파일과 화면에 동시 출력
$ rpm -qa : rpm 패키지를 찾음
$ rpm -qa 이름 : rpm 패키지 내용을 '이름'파일로 저장
$ rpm -qa | tee 이름 : rpm패키지를 저장하면서 화면에도 내용 출력

$ nl 이름 : 빈 행을 제외한 행 번호와 '이름'파일 내용 출력
$ cat -n 이름 : 빈 행을 포함한 행 번호와 '이름'파일 내용 출력
$ nl -w6 이름 : 앞에 6칸 공백을 포함하여 빈 행을 제외한 행 번호와 '이름' 파일 내용 출력
$ nl -i2 이름 : 빈 행을 제외하고 행 번호가 2씩 증가하는 행 번호와 '이름' 파일 내용 출력
$ nl -v10 이름 : 빈 행을 제외하고 행 번호가 10부터 시작하는 행 번호와 '이름' 파일 내용 출력

(ctrl + l : 화면 비우기)
```
